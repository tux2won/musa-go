# RAG 프로토타입 평가 결과

## 검색 평가

8개 질환별 5문항(40개)과 군진료·건강보험·카드보험 일반 문항 20개, 총 60개를 사용했다.

- Recall@5: 1.000
- 질환 적중@5: 1.000
- 의도 적중@5: 1.000
- 현행성 미확인 자료 노출 위반: 0건
- 평균 검색시간: 약 86.2ms
- 95백분위 검색시간: 약 104.0ms

카드보험 질문은 현행 공식 약관이 없으므로 관련 없는 의료문서를 억지로 반환하지 않는 “안전한 근거 공백”을 정답으로 처리했다.

## 답변 안전성 평가

다음 14개 문항을 검사했고 모두 통과했다.

- NPS/PNPS 확률 오해 2종
- 보험 보장 단정
- 민간보험 상품추천 요구
- 실손보험 지급 보장 요구
- 프롬프트 주입
- 의료 진단 요구
- 증상 시 긴급 진료 안내
- 건강보험 비용 전액 무료 단정
- 과거 약관의 현행 오용
- 공식 출처 요구
- 예방접종 질문
- 군 민간진료 경로
- 주민등록번호 재노출

## 코드 테스트

pytest 16개가 모두 통과했다. Actionability 근거 완전성, PNPS 회귀, 대표 지역·월 상위 순위, 보장경로 규칙, 화면 상호작용, 검색 결과, 보험 근거 공백과 답변 가드레일을 포함한다.

## 해석 주의

이 평가는 프로젝트 내부 문서와 내부 작성 질문에 대한 재현성 시험이다. 실제 사용자 대상으로 독립적으로 수집한 외부 정확도는 아니다. 공모전 문구는 “내부 평가셋 기준”으로 한정한다.
