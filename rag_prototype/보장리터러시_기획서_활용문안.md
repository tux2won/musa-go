# 복무온 보장 리터러시 기획서·포트폴리오 활용문안

## 한 문장 소개

복무온은 입영지역과 월별 감염병 통계를 분석해 장병에게 필요한 질환 정보를 우선 제시하고, 공식 군의료·건강보험·공공지원 문서를 연결하여 진료와 비용 확인 경로를 단계별로 안내하는 보장 리터러시 서비스다.

## 문제 정의

막 성인이 되어 입대하는 장병은 감염병 자체뿐 아니라 의무실·군병원·민간병원의 이용 차이, 공무상요양과 건강보험의 관계, 진료비 증빙과 기존 보험 확인 방법을 한 번에 이해하기 어렵다. 정보가 여러 기관과 법령에 흩어져 있어 진료가 늦어지거나, 지원 가능한 비용을 놓치거나, 현행 약관을 확인하지 않은 채 보험 보장을 오해할 수 있다.

## 해결 방식

1. 공공 통계로 복무지역·월에 맞는 감염병 안내순위를 계산한다.
2. 사용자가 현재 상황을 선택하면 군의료→공공보장→민간의료→기존 민간보험 순으로 확인 경로를 제시한다.
3. 구조화 규칙이 보장카드를 결정하고 RAG는 현행 공식 문서의 쉬운 설명과 근거 인용만 담당한다.
4. 영수증·세부내역서·처방전·진단서 등 상황별 서류 체크리스트를 제공한다.
5. 보험상품을 판매하지 않고 보험기간·피보험자·보장범위·면책·청구서류를 스스로 확인하는 법을 교육한다.

## 기대효과

- 감염병 증상 발생 시 진료 지연과 잘못된 의료경로 선택 감소
- 군의료·공공보장·건강보험 제도에 대한 장병의 이해도 향상
- 증빙 누락으로 인한 비용지원·보험청구 실패 예방
- 입대 초기 청년의 보험약관·급여·비급여 이해도 향상
- 통계분석·정책문서 RAG·책임 있는 AI·금융 UX를 결합한 공공서비스 모델 제시

## 사업화·확장 방향

- 병무청 입영안내 또는 나라사랑포털의 입대 전 교육 모듈
- 국방부·군 의료기관의 장병 진료경로 안내 서비스
- 금융교육기관·보험교육 플랫폼의 군 장병 특화 교육 콘텐츠
- 향후 현행 공식 약관 API 또는 보험가입조회 서비스와 연계하되 사용자 동의와 개인정보 최소수집을 전제로 설계

## 성과지표

보험가입 전환율을 핵심지표로 삼지 않는다.

- 안내 전후 군의료·공공보장 경로 이해도 변화
- 상황별 첫 행동 정답률
- 필수 증빙서류 체크리스트 완료율
- 공식 근거 열람률
- 자비진료와 승인 위탁진료 구분 정확도
- 민간보험 보장 단정 응답률 0%

## 발표용 핵심 문장

> AMR은 누구에게, RQ는 어디에서, Seasonality는 언제, Actionability는 무엇을 할 수 있는지를 설명합니다. 이 네 지표가 질환 안내순위를 만들고, 구조화된 보장경로와 RAG가 장병에게 어디로 가고 무엇을 확인하며 어떤 서류를 남겨야 하는지를 알려줍니다.

> 복무온의 맞춤은 개인의 질병을 예측하는 맞춤이 아니라, 복무환경과 진료상황에 따라 필요한 정보의 순서를 바꾸는 책임 있는 상황 맞춤입니다.

## 필수 면책문구

- PNPS는 개인 발병확률·임상적 중증도·보험 보장 가능성이 아니다.
- 공무상 여부, 건강보험 급여, 위탁진료비와 민간보험금은 담당기관의 심사 전 확정할 수 없다.
- 응급하거나 빠르게 악화되는 증상은 비용·보험 확인보다 즉시 진료가 먼저다.
- 서비스는 특정 민간보험 상품을 추천하거나 가입을 중개하지 않는다.
